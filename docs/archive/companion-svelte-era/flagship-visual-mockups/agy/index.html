Tactile greeting hero using Fraunces Display with a breathing gradient orb and vertical list layout of starter prompt chips that pre-populate the input on tap.

Restructured panel featuring human-readable thread titles, active background task rows, settings trigger, and the complete elimination of noisy system console details.

04 — iOS native Overflow Sheet

An iOS-native floating card overlay replacing the always-on action bar. Simulates double-click/long-press behavior with a springy slide-up animation.

06 — Active Work in Thread

The core Sully differentiator preserved: a quiet, high-information worker pill showcasing background processing states without cluttering assistant speech bubble space.

07 — Immersive Voice Interface

A full-screen voice overlay with tabs to toggle between Idle, Listening, and Speaking states. Animates the central gradient orb based on audio phase.

Operational Info

  • Palette: Strictly locked Indigo tokens (no retired magenta).
  • Typography: Blend Mk II (Fraunces Display for titles, Bricolage Grotesque for body/chrome, JetBrains Mono for metadata).
  • Mockup Port: Serve on port 8766 to compare side-by-side with CUR on port 8765.
  • Start Command: python3 -m http.server 8766 --bind 0.0.0.0 from this directory.
