# SVG Assets Directory

This folder contains the production-ready and archived vector icon assets for the Sully OS Work Surface.

## Directory Structure

- **workers/**: SVG icons representing specific LLMs, AI agents, and custom sub-agents (e.g., Gemini, Antigravity, Claude).
- **system/**: SVG icons representing system status, pipelines, routing, queues, approvals, and packet nodes.
- **archive/**: Archive containing older concepts, legacy drawings, and alternative designs. Kept for historical reference.

Please check the master registry in the parent directory for status, purpose, and detailed reviews: [work_surface_asset_registry.md](../work_surface_asset_registry.md).
