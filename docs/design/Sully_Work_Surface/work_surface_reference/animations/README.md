# Animations Directory

This directory is reserved for animation assets, dynamic transitions, and motion scripts (CSS/JS) related to the Sully Work Surface.

## Intended Contents
- **css/**: CSS keyframes and spring transition definitions for micro-animations (e.g., node expansions, path glowing, pulsing packets).
- **js/**: JavaScript-driven dynamic transitions, SVG path offset animators, and graph drawing engines.
- **demos/**: Small isolated animation demos testing visual feedback.
